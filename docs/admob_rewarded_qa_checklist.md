## 1) 빌드/환경

- [x] 확인 / [ ] 미확인 - 프로덕션 빌드 설치 (`eas build --profile production --platform android`)
- [x] 확인 / [ ] 미확인 - EAS 변수 설정 (`EXPO_PUBLIC_ADMOB_REWARD_TYPE=hint_ticket`)
- [x] 확인 / [ ] 미확인 - EAS 변수 설정 (`EXPO_PUBLIC_ADMOB_REWARD_AMOUNT=1`)
- [x] 확인 / [ ] 미확인 - 보상형 광고 유닛 ID가 운영용 ID(테스트 ID 아님)
